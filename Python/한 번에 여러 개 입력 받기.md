# 한 번에 여러 개 입력 받기

## 한 번에 값 2개 입력받기

```python
a, b = input().split() # 문자열 입력
a, b = map(int, input().split()) # 정수 입력
a, b = map(float, input().split()) # 실수 입력
```

## 리스트로 입력받기
```python
# 정수형 리스트로 저장
list1 = list(map(int, input().split()))

# 문자열 n줄 입력받기
list2 = [input() for _ in range(n)]
```
